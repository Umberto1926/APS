## Requisiti
- Python 3.10+
- `cryptography`  (`pip install cryptography`)

Nessun'altra dipendenza: il TLS usa il modulo standard `ssl`.

## Come eseguire la simulazione completa
Dalla cartella che contiene il package `voting/`, cioè quella sopra voting:

    python -m voting.run_demo oppure py -m voting.run_demo 


## Come eseguire i benchmark
    py -m voting.benchmark oppure py -m voting.run_demo
